## Running the Backend

This project uses **Node.js** and **Express** for the backend.

### Prerequisites

Make sure you have **Node.js** installed.

You can check by running:

```bash
node -v
```

If this prints a version number, Node.js is installed.  
Otherwise, download it here: https://nodejs.org/en/download

---

### Step 1: Clone the repository

```bash
git clone https://github.com/nwplus/nwplus-microprojects-pomodoro.git
cd nwplus-microprojects-pomodoro
```

---

### Step 2: Navigate to the backend folder

```bash
cd backend
```

---

### Step 3: Install dependencies

```bash
npm install
```

---

### Step 4: Start the server

```bash
node server.js
```

You should see:

```
Server running on http://localhost:8000
```

---

### Step 5: Verify the server is running

Open your browser and go to:

```
http://localhost:8000
```

If you see a confirmation message, the backend is running
